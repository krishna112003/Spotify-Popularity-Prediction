## 📌 Project Overview

Spotify Music Analytics & Song Popularity Prediction Platform is a premium end-to-end Machine Learning application built using **Python, Streamlit, Scikit-Learn, Plotly, and Pandas**.

The platform analyzes Spotify tracks, explores music trends, compares machine learning models, predicts song popularity scores, and generates AI-powered insights through an interactive dashboard.

<br>

---

# 🚀 Features

### 📊 Interactive Analytics Dashboard

- Dataset Overview
- Music Feature Analysis
- Correlation Matrix
- Trend Analysis
- Popularity Distribution
- Artist Performance Insights

<br>

### 🤖 Machine Learning Pipeline

- Data Preprocessing
- Feature Engineering
- Model Training
- Hyperparameter Tuning
- Model Evaluation
- Prediction System

### 🏆 Model Comparison

The platform compares multiple Machine Learning models:

- AdaBoost Regressor
- Decision Tree Regressor
- Linear Regression
- Ridge Regression
- Lasso Regression
- ElasticNet Regression
- KNN Regressor
- Bagging Regressor
- HistGradientBoosting Regressor
- Voting Regressor
- Stacking Regressor

### 🎯 Prediction Studio

Users can:

- Enter song features manually

- Decade-wise insights
